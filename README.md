# jump-scares

It's like Does The Dog Die, but for escape rooms.

# Contributions

We welcome contributions! Please make a PR. For each PR please say what testing you've done, otherwise your PR will not be accepted.

# Zero to Hero

* Clone this repo
* Get up to date values for all the keys in `.env.template` and copy them over to a new `.env` at root
* Run

```
npm install
npm run dev
```
